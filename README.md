# WizFi630S Test Tool

## Guide

### Use virtual environment

#### If virtualenv not installed

```
pip install virtualenv
```

#### make & activate

```
virtualenv .venv
```

* Windows

  ```
  call .venv\Scripts\activate
  ```

* Linux

  ```
  source .venv/bin/activate
  ```

### Package Install

```
pip install -r requirements.txt
```

### Run

```
python main.py
```
